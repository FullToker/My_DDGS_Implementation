DDGS 代码架构与训练流程                                                                            
                                                                                                     
  项目结构                                                                                           
                                                                                                     
  DDGS/                                                                                              
  ├── train.py                 # 训练入口                                                            
  ├── render.py                # 渲染评估                                                            
  ├── metric.py                # 指标计算                                                            
  ├── arguments/                                                                                     
  │   └── __init__.py          # 参数定义 (ModelParams, OptimizationParams)                          
  ├── scene/                                                                                         
  │   ├── __init__.py          # Scene 类                                                            
  │   ├── dataset_readers.py   # 数据加载 (COLMAP/Blender/Replica)                                   
  │   ├── colmap_loader.py     # COLMAP 文件解析                                                     
  │   ├── cameras.py           # Camera 类                                                           
  │   └── gaussian_model.py    # GaussianModel 类                                                    
  ├── gaussian_renderer/                                                                             
  │   └── __init__.py          # 渲染器 + DD-Drop 机制                                               
  ├── utils/                                                                                         
  │   ├── camera_utils.py      # 相机工具                                                            
  │   ├── loss_utils.py        # 损失函数                                                            
  │   └── ...                                                                                        
  ├── submodules/                                                                                    
  │   ├── diff-gaussian-rasterization/  # CUDA 光栅化                                                
  │   └── simple-knn/                    # KNN 加速                                                  
  └── tools/                                                                                         
      ├── colmap_llff.py       # LLFF 数据准备                                                       
      ├── colmap_360.py        # 360 数据准备                                                        
      └── generate_masks.py    # Mask 生成 (新增)                                                    
                                                                                                     
  ---                                                                                                
  训练流程 (train.py)                                                                                
                                                                                                     
  ┌─────────────────────────────────────────────────────────────┐                                    
  │  1. 参数解析                                                 │                                   
  │     ModelParams: -s, -m, -r, --n_views, --eval              │                                    
  │     OptimizationParams: --depth_weight, --density_weight,   │                                    
  │                         --drop_min, --drop_max, --lambda_far│                                    
  └─────────────────────────────────────────────────────────────┘                                    
                                ↓                                                                    
  ┌─────────────────────────────────────────────────────────────┐                                    
  │  2. 数据加载 (scene/__init__.py + dataset_readers.py)       │                                    
  │                                                             │                                    
  │     Scene(dataset, gaussians)                               │                                    
  │       ├── 检测场景类型 (Colmap/Blender/Replica)              │                                   
  │       ├── readColmapSceneInfo()                             │                                    
  │       │     ├── 读取 cameras.bin → 相机内参                  │                                   
  │       │     ├── 读取 images.bin → 相机外参                   │                                   
  │       │     ├── 读取 {n}_views/dense/fused.ply → 点云       │                                    
  │       │     └── 读取 poses_bounds.npy → 深度范围             │                                   
  │       └── 划分 train/test 相机                              │                                    
  └─────────────────────────────────────────────────────────────┘                                    
                                ↓                                                                    
  ┌─────────────────────────────────────────────────────────────┐                                    
  │  3. 初始化 GaussianModel (scene/gaussian_model.py)          │                                    
  │                                                             │                                    
  │     从点云初始化 3D Gaussian 参数:                           │                                   
  │       - xyz: 位置 (N, 3)                                    │                                    
  │       - features_dc/rest: 球谐系数 (颜色)                    │                                   
  │       - opacity: 不透明度 (N, 1)                            │                                    
  │       - scaling: 缩放 (N, 3)                                │                                    
  │       - rotation: 旋转四元数 (N, 4)                         │                                    
  └─────────────────────────────────────────────────────────────┘                                    
                                ↓                                                                    
  ┌─────────────────────────────────────────────────────────────┐                                    
  │  4. 加载预处理 Mask (如果 lambda_far > 0)                    │                                   
  │                                                             │                                    
  │     load_preprocessed_masks()                               │                                    
  │       路径: ./preprocessed_masks_{mask_param}/{scene}/r{r}/ │                                    
  │       文件: {image_name}.pt → far_mask tensor               │                                    
  └─────────────────────────────────────────────────────────────┘                                    
                                ↓                                                                    
  ┌─────────────────────────────────────────────────────────────┐                                    
  │  5. 训练循环 (10000 iterations)                             │                                    
  │                                                             │                                    
  │     for iteration in range(1, 10001):                       │                                    
  │                                                             │                                    
  │       ┌─────────────────────────────────────────────────┐   │                                    
  │       │ 5.1 随机选择训练视角                              │   │                                  
  │       │     viewpoint_cam = random(train_cameras)        │   │                                   
  │       └─────────────────────────────────────────────────┘   │                                    
  │                          ↓                                  │                                    
  │       ┌─────────────────────────────────────────────────┐   │                                    
  │       │ 5.2 渲染 + DD-Drop (gaussian_renderer/__init__)  │   │                                   
  │       │                                                  │   │                                   
  │       │  render(viewpoint_cam, gaussians, ...,           │   │                                   
  │       │         depth_weight, density_weight,            │   │                                   
  │       │         drop_min, drop_max)                      │   │                                   
  │       │                                                  │   │                                   
  │       │  DD-Drop 机制:                                   │   │                                   
  │       │    1. 计算每个 Gaussian 到相机的深度              │   │                                  
  │       │    2. depth_score = 1 - normalize(depth)         │   │                                   
  │       │    3. density_score = local_density              │   │                                   
  │       │    4. combined = depth_w * depth + density_w *   │   │                                   
  │       │                  density                         │   │                                   
  │       │    5. drop_prob = lerp(drop_min, drop_max,       │   │                                   
  │       │                        combined)                 │   │                                   
  │       │    6. 随机 dropout 高分 Gaussian                  │   │                                  
  │       └─────────────────────────────────────────────────┘   │                                    
  │                          ↓                                  │                                    
  │       ┌─────────────────────────────────────────────────┐   │                                    
  │       │ 5.3 计算损失                                      │   │                                  
  │       │                                                  │   │                                   
  │       │  L1_loss = |render - gt|                         │   │                                   
  │       │  SSIM_loss = 1 - SSIM(render, gt)                │   │                                   
  │       │  loss = L1 + λ_dssim * SSIM_loss                 │   │                                   
  │       │                                                  │   │                                   
  │       │  DAFE (如果 lambda_far > 0):                     │   │                                   
  │       │    far_loss = |render * far_mask - gt * far_mask││   │                                   
  │       │    loss += λ_far * far_loss                      │   │                                   
  │       └─────────────────────────────────────────────────┘   │                                    
  │                          ↓                                  │                                    
  │       ┌─────────────────────────────────────────────────┐   │                                    
  │       │ 5.4 反向传播 + 优化器更新                         │   │                                  
  │       │     loss.backward()                              │   │                                   
  │       │     optimizer.step()                             │   │                                   
  │       └─────────────────────────────────────────────────┘   │                                    
  │                          ↓                                  │                                    
  │       ┌─────────────────────────────────────────────────┐   │                                    
  │       │ 5.5 自适应密度控制 (每100次)                      │   │                                  
  │       │     - 克隆: 梯度大 + 尺寸小的 Gaussian            │   │                                  
  │       │     - 分裂: 梯度大 + 尺寸大的 Gaussian            │   │                                  
  │       │     - 剪枝: 不透明度低的 Gaussian                 │   │                                  
  │       └─────────────────────────────────────────────────┘   │                                    
  │                                                             │                                    
  └─────────────────────────────────────────────────────────────┘                                    
                                ↓                                                                    
  ┌─────────────────────────────────────────────────────────────┐                                    
  │  6. 保存模型                                                 │                                   
  │     output/{model}/point_cloud/iteration_10000/             │                                    
  └─────────────────────────────────────────────────────────────┘

  核心创新                                                                                           
  ┌─────────┬──────────────────────────────────────┬─────────────────────────────────────┐           
  │  机制   │                 位置                 │                作用                 │           
  ├─────────┼──────────────────────────────────────┼─────────────────────────────────────┤           
  │ DD-Drop │ gaussian_renderer/__init__.py:90-121 │ 基于深度+密度的 Dropout，抑制过拟合 │           
  ├─────────┼──────────────────────────────────────┼─────────────────────────────────────┤           
  │ DAFE    │ train.py:143-148                     │ 远距离区域增强监督，避免欠拟合      │           
  └─────────┴──────────────────────────────────────┴─────────────────────────────────────┘           
  ---                                                                                                
  关键参数                                                                                           
  ┌──────────────────┬────────┬──────────────────────────┐                                           
  │       参数       │ 默认值 │           作用           │                                           
  ├──────────────────┼────────┼──────────────────────────┤                                           
  │ --depth_weight   │ 0.3    │ DD-Drop 中深度因子权重   │                                           
  ├──────────────────┼────────┼──────────────────────────┤                                           
  │ --density_weight │ 0.7    │ DD-Drop 中密度因子权重   │                                           
  ├──────────────────┼────────┼──────────────────────────┤                                           
  │ --drop_min       │ 0.05   │ Dropout 最小概率         │                                           
  ├──────────────────┼────────┼──────────────────────────┤                                           
  │ --drop_max       │ 0.3    │ Dropout 最大概率         │                                           
  ├──────────────────┼────────┼──────────────────────────┤                                           
  │ --lambda_far     │ 1.0    │ DAFE 远距离增强权重      │                                           
  ├──────────────────┼────────┼──────────────────────────┤                                           
  │ --mask_param     │ 10     │ 远距离区域阈值 (最远 N%) │                                           
  └──────────────────┴────────┴──────────────────────────┘

  数据要求                                                                                           
                                                                                                     
  data/                                                                                              
  ├── images/*.jpg                      # RGB 图像                                                   
  ├── sparse/0/

  │   └── images.bin                    # 相机外参                                                   
  ├── {n_views}_views/dense/fused.ply   # 点云                                                       
  ├── poses_bounds.npy                  # 深度范围 [N, 17]                                           
  └── (外部) preprocessed_masks_{mask_param}/{scene}/r{r}/*.pt
